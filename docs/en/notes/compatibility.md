# Compatibility of MMYOLO

## MMYOLO 0.3.0

To unify with other OpenMMLab repositories, change all keys of `METAINFO` in Dataset from upper case to lower case.

| Before v0.3.0 | after v0.3.0 |
| :-----------: | :----------: |
|    CLASSES    |   classes    |
|    PALETTE    |   palette    |
| DATASET_TYPE  | dataset_type |
